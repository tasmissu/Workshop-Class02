4. Commit และ Push ขึ้น GitHub

# Step 1 — Fork Repository

Fork repository นี้ไปยัง account ของตัวเอง

> ⚠ Repo ต้องเป็น Public
> ไม่งั้น Boss จะมองไม่เห็นผลงานของเอ็ง

---

# Step 2 — Clone Repository

Clone fork ของตัวเองลงเครื่อง

```bash
git clone <your-fork-url>
```

จากนั้นเข้า folder:

```bash
cd Abyss-Beast-Protocol
```

---

# Step 3 — Create Your Underling

สร้างไฟล์ใหม่ชื่อ:

```txt
<YOUR_MONSTER_NAME_HERE>.md
```

ตัวอย่าง:

```txt
Abyss-Gazer.md
```

จากนั้นก็ลงมือสร้าง ปีศาจร้าย (THE A.B.O.M.I.N.A.T.I.O.N) ของพวกเอ็งได้เลย 

```txt
MONSTER NAME

head:
body:
weapon:
skill:

+ เพิ่มอะไรก็ได้ตามใจเลย
```

ตัวอย่าง:

```txt
THE ABYSS GAZER

head: มิกกี้เมาส์
body: Mike Tyson
weapon: เดนจิตอนแปลงร่างเป็น Chainsaw Man
skill: ตีลังกาได้ 3 ตลบ

special ability: เล่นหมากรุกชนะ Magnus Carlsen(หนึ่งในแชมป์โลกหมากรุก) มามากกว่า 1000 ตา 
weakness: ไม่ชอบตื่นไปโรงเรียนตอนเช้า
Quote ประจำตัว: The Abyss GAZES BACK AT THOSE WHO DARE TO DEFY ME... ARE YOU READY TO BE DEVOUR ?
```

เพิ่มได้หมด:

* Rank
* Lore
* ธาตุ
* จุดอ่อน
* Ultimate Skill
* Quote ประจำตัว

# Step 4 — Check Status

ตรวจสอบว่า Git ตรวจพบไฟล์ใหม่แล้ว

```bash
git status
```

---

# Step 5 — Add File

เพิ่มไฟล์เข้า staging area

```bash
git add <YOUR_MONSTER_NAME_HERE>.md
```

หรือ

```bash
git add .
```

---

# Step 6 — Commit

Commit พร้อม message ที่คนอ่านแล้วรู้ว่าสร้าง ปิ ศ า จ ร้ า ย อ ะ ไ ร ขึ้ น ม า

ตัวอย่าง:

```bash
git commit -m "FINALIZED: create the Abyss-Gazer"
```

---

# Step 7 — Push

ส่งสมุนขึ้นสู่โลกออนไลน์

```bash
git push
```

---

# Step 8 — Submit

ส่ง comment มาใน Teams ตาม format นี้:

```txt
<your-name> <your-repo-url>
```

ตัวอย่าง:

```txt
John https://github.com/john/Abyss-Beast-Protocol
```
